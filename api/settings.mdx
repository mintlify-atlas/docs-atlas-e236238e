All REST Framework settings are namespaced under the `REST_FRAMEWORK` dictionary in your Django settings.

```python
# settings.py
REST_FRAMEWORK = {
    'DEFAULT_RENDERER_CLASSES': [
        'rest_framework.renderers.JSONRenderer',
    ],
    'DEFAULT_PARSER_CLASSES': [
        'rest_framework.parsers.JSONParser',
    ],
}
```

## API Policy Settings

Settings that control basic API policies applied to all views.

### DEFAULT_RENDERER_CLASSES

<ParamField path="DEFAULT_RENDERER_CLASSES" type="list">
  List of renderer classes for returning responses.
  
  Default:
  ```python
  [
      'rest_framework.renderers.JSONRenderer',
      'rest_framework.renderers.BrowsableAPIRenderer',
  ]
  ```
</ParamField>

### DEFAULT_PARSER_CLASSES

<ParamField path="DEFAULT_PARSER_CLASSES" type="list">
  List of parser classes for parsing request data.
  
  Default:
  ```python
  [
      'rest_framework.parsers.JSONParser',
      'rest_framework.parsers.FormParser',
      'rest_framework.parsers.MultiPartParser'
  ]
  ```
</ParamField>

### DEFAULT_AUTHENTICATION_CLASSES

<ParamField path="DEFAULT_AUTHENTICATION_CLASSES" type="list">
  List of authentication classes.
  
  Default:
  ```python
  [
      'rest_framework.authentication.SessionAuthentication',
      'rest_framework.authentication.BasicAuthentication'
  ]
  ```
</ParamField>

### DEFAULT_PERMISSION_CLASSES

<ParamField path="DEFAULT_PERMISSION_CLASSES" type="list">
  List of permission classes. All classes must grant permission.
  
  Default:
  ```python
  [
      'rest_framework.permissions.AllowAny',
  ]
  ```
</ParamField>

### DEFAULT_THROTTLE_CLASSES

<ParamField path="DEFAULT_THROTTLE_CLASSES" type="list">
  List of throttle classes.
  
  Default: `[]`
</ParamField>

### DEFAULT_CONTENT_NEGOTIATION_CLASS

<ParamField path="DEFAULT_CONTENT_NEGOTIATION_CLASS" type="str">
  Content negotiation class for selecting renderers.
  
  Default: `'rest_framework.negotiation.DefaultContentNegotiation'`
</ParamField>

### DEFAULT_METADATA_CLASS

<ParamField path="DEFAULT_METADATA_CLASS" type="str">
  Metadata class for OPTIONS requests.
  
  Default: `'rest_framework.metadata.SimpleMetadata'`
</ParamField>

### DEFAULT_VERSIONING_CLASS

<ParamField path="DEFAULT_VERSIONING_CLASS" type="str">
  Versioning class for API versioning.
  
  Default: `None`
</ParamField>

### DEFAULT_SCHEMA_CLASS

<ParamField path="DEFAULT_SCHEMA_CLASS" type="str">
  Schema class for automatic schema generation.
  
  Default: `'rest_framework.schemas.openapi.AutoSchema'`
</ParamField>

## Encoding Settings

### UNICODE_JSON

<ParamField path="UNICODE_JSON" type="bool">
  Render JSON with unicode characters instead of escaped sequences.
  
  Default: `True`
</ParamField>

### COMPACT_JSON

<ParamField path="COMPACT_JSON" type="bool">
  Render compact JSON (no extra whitespace).
  
  Default: `True`
</ParamField>

### STRICT_JSON

<ParamField path="STRICT_JSON" type="bool">
  Use strict JSON encoding (reject NaN and Infinity).
  
  Default: `True`
</ParamField>

### COERCE_DECIMAL_TO_STRING

<ParamField path="COERCE_DECIMAL_TO_STRING" type="bool">
  Render Decimal values as strings instead of floats.
  
  Default: `True`
</ParamField>

### COERCE_BIGINT_TO_STRING

<ParamField path="COERCE_BIGINT_TO_STRING" type="bool">
  Render large integers (>53 bits) as strings to prevent precision loss in JavaScript.
  
  Default: `False`
</ParamField>

### UPLOADED_FILES_USE_URL

<ParamField path="UPLOADED_FILES_USE_URL" type="bool">
  Use URL representation for file fields instead of file name.
  
  Default: `True`
</ParamField>

## Throttling Settings

### DEFAULT_THROTTLE_RATES

<ParamField path="DEFAULT_THROTTLE_RATES" type="dict">
  Default throttle rates for throttle classes.
  
  Default:
  ```python
  {
      'user': None,
      'anon': None,
  }
  ```
  
  Rates use the format `'number/period'`, e.g.:
  - `'100/hour'`
  - `'1000/day'`
  - `'5/second'`
</ParamField>

### NUM_PROXIES

<ParamField path="NUM_PROXIES" type="int">
  Number of proxies in front of the API for throttling IP address detection.
  
  Default: `None`
</ParamField>

## Accessing Settings

Access settings using the `api_settings` object:

```python
from rest_framework.settings import api_settings

print(api_settings.DEFAULT_AUTHENTICATION_CLASSES)
print(api_settings.PAGE_SIZE)
```

## Example Configuration

```python
# settings.py
REST_FRAMEWORK = {
    # Renderers
    'DEFAULT_RENDERER_CLASSES': [
        'rest_framework.renderers.JSONRenderer',
    ],
    
    # Parsers
    'DEFAULT_PARSER_CLASSES': [
        'rest_framework.parsers.JSONParser',
    ],
    
    # Authentication
    'DEFAULT_AUTHENTICATION_CLASSES': [
        'rest_framework_simplejwt.authentication.JWTAuthentication',
    ],
    
    # Permissions
    'DEFAULT_PERMISSION_CLASSES': [
        'rest_framework.permissions.IsAuthenticated',
    ],
    
    # Pagination
    'DEFAULT_PAGINATION_CLASS': 'rest_framework.pagination.PageNumberPagination',
    'PAGE_SIZE': 100,
    
    # Filtering
    'DEFAULT_FILTER_BACKENDS': [
        'django_filters.rest_framework.DjangoFilterBackend',
        'rest_framework.filters.SearchFilter',
        'rest_framework.filters.OrderingFilter',
    ],
    
    # Throttling
    'DEFAULT_THROTTLE_CLASSES': [
        'rest_framework.throttling.AnonRateThrottle',
        'rest_framework.throttling.UserRateThrottle',
    ],
    'DEFAULT_THROTTLE_RATES': {
        'anon': '100/hour',
        'user': '1000/hour',
    },
    
    # Versioning
    'DEFAULT_VERSIONING_CLASS': 'rest_framework.versioning.NamespaceVersioning',
    'DEFAULT_VERSION': 'v1',
    'ALLOWED_VERSIONS': ['v1', 'v2'],
    
    # Schema
    'DEFAULT_SCHEMA_CLASS': 'rest_framework.schemas.openapi.AutoSchema',
    
    # Date/Time formats
    'DATETIME_FORMAT': '%Y-%m-%d %H:%M:%S',
    'DATE_FORMAT': '%Y-%m-%d',
    
    # Exception handling
    'EXCEPTION_HANDLER': 'myapp.exceptions.custom_exception_handler',
    
    # Testing
    'TEST_REQUEST_DEFAULT_FORMAT': 'json',
}
```
